## ✅ 로컬 테스트 (개발 PC)

### EXE 직접 실행 테스트
- [ ] `dist\AutoFM.exe` 더블클릭 실행
- [ ] 애플리케이션 창이 정상적으로 열림
- [ ] 데이터베이스 파일 생성 확인
  - 위치: `%LOCALAPPDATA%\auto_fm\app.db`
  ```bash
  dir %LOCALAPPDATA%\auto_fm
  ```

### 기능 테스트
- [ ] 노무비 입력 및 계산
  - 직무 추가
  - 인원 및 근무일 입력
  - 계산 결과 확인
- [ ] 경비 입력 및 계산
  - 경비 항목 추가
  - 수량 및 단가 입력
- [ ] 일반관리비/이윤 검증
  - 10% 요율로 계산 확인
  - 15% 입력 시 10%로 캡핑 확인
- [ ] 시나리오 저장
  - JSON 파일 생성 확인
  - 파일명 및 위치 확인
- [ ] 시나리오 로드
  - 저장한 파일 불러오기
  - 데이터 복원 확인
- [ ] PDF 내보내기
  - 요약 PDF 생성
  - 상세 PDF 생성
  - 한글 표시 확인
- [ ] Excel 내보내기
  - 다중 시트 확인
  - 데이터 정확성 확인
- [ ] 시나리오 비교
  - 2개 이상 시나리오 비교
  - 차이값 계산 확인

### 안정성 테스트
- [ ] 잘못된 입력 처리
  - 음수 입력
  - 문자 입력
  - 빈 값
- [ ] 대용량 데이터
  - 많은 직무 (10개 이상)
  - 많은 경비 항목 (20개 이상)
- [ ] 연속 작업
  - 여러 시나리오 생성
  - 반복 계산
  - 메모리 누수 확인 (작업 관리자)
